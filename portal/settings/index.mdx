## Mein Account

Unter **Einstellungen → Mein Account** können persönliche Daten und Zugriffsrechte verwaltet werden.

### Profil

Name und E-Mail-Adresse lassen sich jederzeit aktualisieren. Die E-Mail-Adresse wird für den Login und alle Benachrichtigungen verwendet.

### Passwort ändern

Das Passwort kann über **Einstellungen → Mein Account → Passwort ändern** aktualisiert werden. Es wird das aktuelle Passwort sowie zweimal das neue Passwort benötigt.


### Einladungen

Unter **Einstellungen → Mein Account → Einladungen** werden alle ausstehenden Einladungen angezeigt, die der eigene Account von anderen Organisationen erhalten hat. Von hier aus kann eine Einladung angenommen oder abgelehnt werden.

---

## Organisation

Unter **Einstellungen → Organisation** werden organisationsweite Einstellungen verwaltet. Diese sind für alle Mitglieder sichtbar und wirken sich auf die gesamte Nutzung aus.

### Name & ID

Der Organisationsname und die Organisations-ID können hier angepasst werden. Die ID wird für interne Referenzen und API-Aufrufe verwendet.

### Mitglieder

Alle aktiven Mitglieder der Organisation werden hier aufgelistet. Bestehende Mitglieder können entfernt und deren Rollen angepasst werden.

### Integrationen

Unter **Einstellungen → Organisation → Integrationen** lassen sich externe Dienste verbinden, z. B. Whatsapp-Notification. Einmal konfiguriert, stehen die Integrationen allen Mitgliedern der Organisation zur Verfügung.

### Abrechnung

Unter **Einstellungen → Organisation → Abrechnung** können der aktuelle Plan eingesehen, die Zahlungsmethode geändert und das Abonnement verwaltet werden.

Siehe: [Abonnement & Pläne](/portal/abonnement/index)

## Häufige Fragen

<AccordionGroup>
  <Accordion title="Wo sehe ich Einladungen, die ich erhalten habe?">
    Einladungen von anderen Organisationen werden unter Einstellungen → Mein Account → Einladungen angezeigt. Dort können sie angenommen oder abgelehnt werden.
  </Accordion>
  <Accordion title="Kann ich meine E-Mail-Adresse ändern?">
    Ja. Die E-Mail-Adresse lässt sich unter Einstellungen → Mein Account anpassen. Sie wird für Login und Benachrichtigungen verwendet.
  </Accordion>
  <Accordion title="Was ist die Organisations-ID?">
    Die Organisations-ID ist eine eindeutige Kennung für die Organisation innerhalb von Osiro. Sie wird für API-Integrationen und interne Referenzen genutzt und kann unter Einstellungen → Organisation eingesehen werden.
  </Accordion>
</AccordionGroup>
