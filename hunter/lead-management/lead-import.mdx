---
title: "Lead-Import"
description: "Wie importiere ich meine Leads?"
---

<Steps>
  <Step title="Import starten">
    Wechseln Sie in das Kontrollzentrum von Hunter und klicken Sie oben rechts auf "Import Lead"

    <Frame>
    <img src="/images/hunter/lead-management/lead-import/import-button.png" alt="Kontrollzentrum" />

  </Frame>
    
  </Step>

  <Step title="CSV-Datei importieren">
    Ziehen Sie Ihre CSV-Datei per Drag-and-Drop in das Fenster
    
  </Step>

  <Step title="Spalten anpassen">
    Osiro erkennt Ihre Spalten automatisch. Bei Bedarf können Sie Ihre Spalten aber noch anpassen

    <Tip>
      Osiro setzt beim Import automatisch ein Tag mit dem aktuellen Datum. Sie können dieses Tag anpassen, ein anderes wählen oder ein neues erstellen. Alternativ lässt sich auch eine Spalte Ihrer CSV-Datei als Tag verwenden – weisen Sie ihr dazu die Destination „Tag (per row)" zu.
    </Tip>

    <Frame>
    <img src="/images/hunter/lead-management/lead-import/import-schema.png" />

  </Frame>

  </Step>

  <Step title="Leads importieren">
    Klicken Sie auf "Create Schema & Import Leads" und anschließend auf "Back to Leads". Ihre Leads wurden jetzt erfolgreich importiert.

  </Step>
</Steps>

<AccordionGroup>
  <Accordion title="Welches Dateiformat wird für den Import unterstützt?">
    Der Lead-Import funktioniert ausschließlich mit CSV-Dateien. Stellen Sie sicher, dass Ihre Datei im `.csv`-Format vorliegt, bevor Sie sie per Drag-and-Drop in das Import-Fenster ziehen.
  </Accordion>
  <Accordion title="Was passiert, wenn Osiro meine Spalten falsch erkennt?">
    Osiro erkennt die Spalten Ihrer CSV-Datei automatisch. Sollte die Zuordnung nicht stimmen, können Sie die Spalten im Schritt „Spalten anpassen" manuell korrigieren, bevor Sie den Import abschließen.
  </Accordion>
  <Accordion title="Wie kann ich importierte Leads später wiederfinden?">
    Sie können beim Import Tags setzen – entweder für die gesamte Liste oder über eine dedizierte Spalte in Ihrer CSV-Datei. So lassen sich die importierten Leads anschließend gezielt filtern und wiederfinden.
  </Accordion>
</AccordionGroup>
