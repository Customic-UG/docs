<Note>Dieser Design-Bereich ist identisch für Funnel Editor und Landing Page Editor.</Note>

## Überblick

Im Design-Bereich legen Sie globale Einstellungen fest, die sich auf alle Elemente auswirken.

<img src="/images/convert/design/element-settings.png"/>

<Tip>Eine hier erstellte Konfiguration können Sie anschließend im [Styleguide](/convert/design/styleguide) unter einem eigenen Namen speichern, um sie auch in anderen Funnels oder Landing Pages wiederzuverwenden.</Tip>

---

## Einstellungen

| Einstellung | Optionen | Beschreibung |
|---|---|---|
| **Rahmenrundung** | Eckig / Abgerundet / Rund | Bestimmt, wie stark die Ecken von Elementen abgerundet werden. |
| **Rahmendicke** | Dünn / Normal / Dick | Bestimmt die Dicke des Rahmens von Elementen. |
| **Elementgröße** | Klein / Normal / Groß | Bestimmt die Größe der Elemente. |

### Eingabefeld-Verhalten

Über die Option **Label als Platzhalter verwenden** legen Sie fest, ob das Label eines Eingabefelds als Platzhaltertext innerhalb des Felds angezeigt wird, statt als separate Beschriftung darüber.
